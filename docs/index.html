חקר חישובי של ארכיאולוגיה ומקורות היסטוריים תשפ״ה

מטרות הקורס

לימוד עקרונות ושיטות המחקר החישובי באמצעות שפת התכנות R ופלטפורמת GitHub עבור נתונים ארכיאולוגיים והיסטוריים. הסטודנטים ילמדו ליצור פרויקטי מחקר בעלי יכולת שחזור (reproducible research) ולפרסמם באופן דיגיטלי. הקורס יתמקד ביצירת דוחות מחקר שיתבססו על נתוני פאוסט וספראי אודות דפוסי התיישבות בארץ ישראל, תוך שימוש בכלים לניתוח, ויזואליזציה, והנגשת מידע מחקרי.

תוכן הקורס

הקורס מציג גישה חדשנית למחקר ארכיאולוגי והיסטורי באמצעות פיתוח מיומנויות דיגיטליות והתנסות בשיטות עבודה חישוביות. כיצד ניתן לשפר את שקיפות ושחזוריות המחקר הארכיאולוגי וההיסטורי? איך אפשר לנצל נתוני (עתק) מחפירות ארכיאולוגיות ומקורות היסטוריים? וכיצד ניתן לתקשר ממצאים מחקריים באופן יעיל ומשכנע באמצעות פרסומים נתונים מקוון, גרפים, תרשימים ומפות? סמינר מתודולוגי זה יתמקד בשלושה נושאים מרכזיים: (1) יצירת פרויקט מחקר בעלי יכולת שחזור באמצעות שפת התכנות R וגיטהאב, (2) איסוף וניתוח נתונים היסטוריים וארכיאולוגיים באמצעות טכניקות מתקדמות, ו-(3) תקשור ופרסום ממצאי המחקר באמצעות אתרי Quarto GitHub. הסטודנטים יתנסו במקרה בוחן של ניתוח מסד הנתונים של פאוסט וספראי אודות דפוסי התיישבות בארץ ישראל, וייצרו פרויקט מחקר אישי המבוסס על נתונים אלו. הקורס משלב הן היבט ביקורתי שילמד שיטות חישוביות לקריאה קרובה וניתוח תרבותי אנליטי, והן פרספקטיבה יצירתית שתצייד את התלמידים בכלים לעצב שיטות חדשות של ניתוח וייצוג נתונים. באמצעות חבילות R כגון rrtools, rvest, tidyverse, ggplot, ו-quarto, הסטודנטים ילמדו ליצור פרויקטים מחקריים מלאים מהאיסוף והניתוח של הנתונים ועד הפרסום הדיגיטלי של התוצאות.

מהלך הקורס

  • מכיוון שהקורס מתבצע במסגרת מעבדה, הנוכחות בקורס היא חובה.

  • התקשורת בקורס בין הסטודנטים ובין הסטודנטים למרצה תתנהל בקבוצה ייעודית בדיסקורד (Discord). על התלמידים להרשם לאפליקציה ולאחר מכן להירשם לקבוצה דרך הלינק הבא לפני השיעור הראשון. מטרתה העיקרית של הקבוצה הינה להיות פורום למטרות debugging ושאלות ביאור. על התלמידים להיות פעילים בקבוצה ולעזור אחד לשני כחלק מהשתתפות פעילה. 

  • על כל התלמידים להתקין לפני השיעור הראשון את שפת התכנות R ואת סביבת הפיתוח RStudio, ולפתוח חשבון GitHub אם אין להם עדיין. כמו כן, חשוב להרשם ל-OpenAI כדי להפיק API עבור ChatGPT. להסבר מפורט על התקנת R ו-RStudio יש לגשת ללינק הזה ומי שרוצה יש גם הסבר בסרטוני התקנה חלק 1 ו-חלק 2 (יש תרגום אוטומטי לכתוביות לעברית). בשיעור הראשון נתקין את החבילות הנדרשות ונלמד כיצד להגדיר את GitHub באופן מתאים. לבעיות התקנה, אנא כתבו הודעה בקבוצת ה-discord.

  • במהלך הסמסטר יהיו תרגילים שבועיים להבנת הנושא, הקוד, והשיטות החישוביות השונות. בסוף כל יחידה יגישו התלמידים תרגיל מסכם, סה״כ 3 - מסומנים מטה, לפירוט התרגילים והקריאה לכל שיעור יש לעיין בתכנית ההוראה מטה.

  • פרויקט הגמר של הקורס הוא דוח מחקרי המפורסם באתר Quarto GitHub אישי. כל סטודנט יבחר נושא מחקר מתוך מסד הנתונים של חפירות א״י, יחקור אותו באמצעות R, ויפרסם את התוצאות באתר אינטרנט מקצועי. בתאריך 15 ביוני יתקיימו הצגות של הפרויקטים בזום, כל הצגה 10 דק׳ וזמן קצר לדיון.
